Privacy Policy for LLM Prompt Saver Last updated: July 28, 2024 Information Collection: LLM Prompt Saver collects and stores user-created text prompts for various Language Learning Models (LLMs). Use of Information: The collected prompts are used solely to provide the core functionality of the extension - allowing users to save and reuse prompts for multiple LLMs. Data Storage: All data (prompts) is stored locally in the user's browser using Chrome's storage API. No data is transmitted or stored on external servers. Data Access: The saved prompts are accessible only to the user on their local device. The developers of LLM Prompt Saver do not have access to any user data. Data Control: Users can delete their saved prompts at any time within the extension. All data is automatically deleted when the extension is uninstalled. Updates to Privacy Policy: We may update this privacy policy from time to time. We will notify users of any changes by posting the new privacy policy on this page. Contact: If you have any questions about this privacy policy, please contact us at: ryanlow@gmail.com
